## 官网下载

Node.js 各平台下载地址（含npm）： https://nodejs.org/en/download/

npm 官网： https://www.npmjs.com/get-npm


## 查看版本和更新

查看 node 和 npm 版本
```
    node -v

	npm -v
```

更新 npm 版本
```
	npm install npm@latest -g
```

## 安装淘宝镜像

安装淘宝镜像的包命令行管理工具cnpm
```
	npm install cnpm -g --registry=http://registry.npm.taobao.org
```
